# Advaya 🔒
**Fully Decentralized, E2E Encrypted Web3 Messenger on the Stellar Network.**

Advaya is a next-generation decentralized application (dApp) that provides fully secure, end-to-end encrypted messaging. By leveraging **Waku** for peer-to-peer message routing, **IPFS** for decentralized file/payload storage, and the **Stellar & Soroban Network** for wallet registration, profile discovery, and spam prevention, Advaya ensures your communications remain private, censorship-resistant, and entirely under your control.

## 🚀 Future Roadmap & Evolution
Based on the collected user feedback and observations during the Level 5 validation phase, we plan to implement the following improvements in the next development cycle:

1. **Robust Anti-Spam (Implemented):** Required a 1 XLM payment via Freighter when starting a new chat to deter network spam (`e74e398`, `6b58bfc`).
2. **Resilient Decentralized Networking (Implemented):** Added a 3-second timeout to Waku sends to prevent freezing when privacy browsers block peer discovery, instantly switching to fallback nodes (`9c4309a`).
3. **Hybrid Caching Protocol:** Build upon the implemented local caching (`6478905`) to create a fully offline-first experience that automatically syncs encrypted IPFS payloads when re-connected.
4. **Soroban Smart Contract Extension:** Upgrade the newly implemented Soroban wallet and conversation manager (`d293438`) to allow for dynamic, user-configured fee thresholds (e.g. charging 5 XLM instead of 1 XLM to chat with prominent users).

## ✨ Advanced Features 🌟
1. **Fully Decentralized Architecture:** Utilizes the Waku protocol for message broadcasting (LightPush/Filter) eliminating centralized points of failure.
2. **E2E Encryption:** All messages are encrypted locally in the browser using `tweetnacl` (X25519) before being stored on IPFS or broadcast over Waku.
3. **Freighter Wallet Anti-Spam Gate:** A built-in transaction builder that automatically prompts users to sign a 1 XLM payment on the Stellar network to initiate a peer-to-peer chat.
4. **Soroban Contract Registry:** Smart contracts on the Stellar testnet handle public key distribution and state tracking.

## 🚀 Getting Started (Local Development)

### Prerequisites
- Node.js (v18+)
- Freighter Wallet browser extension
- Pinata JWT for IPFS

### 1. Clone the repository
```bash
git clone https://github.com/rajkumarsharma316/Advaya.git
cd Advaya
```

### 2. Smart Contract Setup
```bash
cd contracts/advaya
cargo build --target wasm32-unknown-unknown --release
stellar contract deploy --wasm target/wasm32-unknown-unknown/release/advaya.wasm --source deployer --network testnet
```

### 3. Frontend Setup
```bash
cd apps/web
npm install
```
Create a `.env.local` file based on the setup:
```env
NEXT_PUBLIC_STELLAR_NETWORK=testnet
NEXT_PUBLIC_SOROBAN_CONTRACT_ID=your_contract_id
NEXT_PUBLIC_PINATA_GATEWAY=https://gateway.pinata.cloud/ipfs
NEXT_PUBLIC_PINATA_JWT=your_jwt
NEXT_PUBLIC_TREASURY_ADDRESS=your_stellar_wallet
```
Start the frontend:
```bash
npm run dev
```
